JavaScript, often abbreviated as JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. As of 2022, 98% of websites use JavaScript on the client side for webpage behavior, often incorporating third-party libraries.

  • Experienced in using JavaScript libraries and frameworks, such as jQuery, React, and Angular, to streamline development and improve performance
  • Proficient in object destructuring and arrow functions for more concise and readable code.
  • Knowledgeable in synchronous and asynchronous error handling to identify and resolve issues in programs
Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured (particularly procedural), object-oriented and functional programming. It is often described as a "batteries included" language due to its comprehensive standard library.
  • Proficient in Python programming language and its object-oriented programming methods
  • Experienced in using built-in Python functions and modules to manipulate strings, lists, and other data types
  • Skilled in creating custom functions and classes to perform complex tasks and add functionality to programs
React is an open-source JavaScript library that is used for building user interfaces (UIs). React allows developers to build reusable UI components and efficiently update the UI in response to changes in application state, without having to manually manipulate the DOM.
  • Proficient in React library and its component-based architecture
  • Experienced in creating and managing stateful and stateless React components
  • Skilled in using React hooks to manage component state and perform side effects
